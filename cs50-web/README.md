# CS50W - Web Programming with Python and JavaScript

## Course Overview
CS50 Web picks up where CS50x leaves off, diving more deeply into the design and implementation of web applications with Python, JavaScript, and SQL using frameworks like Django, React, and Bootstrap. This course emphasizes scalability, security, and user experience.

- **Duration**: 12 weeks
- **Time Commitment**: 10-20 hours per week
- **Difficulty**: Intermediate
- **Prerequisites**: CS50x or equivalent programming experience

### Week 7: Testing, CI/CD
- Quality assurance and deployment
- **Topics**: Unit tests, Selenium, GitHub Actions, Docker
- **Project**: Automated testing pipeline

- **Topics**: Load balancing, caching, database optimization

- **Topics**: HTTPS, authentication, CSRF, XSS prevention

- Application Programming Interfaces
- **Topics**: REST APIs, GraphQL, API design

- Capstone project development
- **Requirements**: Original web application using course concepts
