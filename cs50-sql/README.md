# CS50 SQL - Introduction to Databases with SQL

Introduction to databases using SQL.

- **Week 0**: Querying
- **Week 1**: Relating

- **Week 4**: Viewing
- **Week 5**: Optimizing
- **Week 6**: Scaling

## Key Topics

SELECT, JOIN, database design, indexes, query optimization

SQLite, PostgreSQL, MySQL
