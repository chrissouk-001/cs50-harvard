# CS50 SQL - Introduction to Databases with SQL

## Course Overview
CS50 SQL is an introduction to databases using Structured Query Language (SQL). Learn how to create, read, update, and delete data with relational databases, model real-world entities, and optimize database performance. The course covers SQLite, PostgreSQL, and MySQL.

## üìä SQL Concepts Covered

### Basic Queries
- **SELECT**: Data retrieval
- **WHERE**: Filtering conditions
- **ORDER BY**: Sorting results
- **LIMIT**: Result set limitation
- **DISTINCT**: Unique values

### Advanced Queries
- **JOIN**: Table relationships
- **UNION**: Combining queries
- **Subqueries**: Nested queries
- **Common Table Expressions (CTEs)**
- **Window functions**: Advanced analytics

### Data Manipulation
- **INSERT**: Adding new data
- **UPDATE**: Modifying existing data
- **DELETE**: Removing data
- **REPLACE**: Insert or update operations

### Database Design
- **Tables**: Structure definition
- **Constraints**: Data integrity
- **Indexes**: Performance optimization
- **Views**: Virtual tables
- **Triggers**: Automated actions

## üèóÔ∏è Database Design Principles

### Normalization
- **First Normal Form (1NF)**: Atomic values
- **Second Normal Form (2NF)**: Partial dependency elimination
- **Third Normal Form (3NF)**: Transitive dependency elimination
- **Denormalization**: Performance considerations

### Entity-Relationship Modeling
- **Entities**: Real-world objects
- **Attributes**: Object properties
- **Relationships**: Object connections
- **Cardinality**: Relationship constraints
